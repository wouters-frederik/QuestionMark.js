QuestionMark.js

A lightweight, cross-browser, modal window that displays shortcut keys for your app, similar to Twitter, Gmail, GitHub, etc.

This is an example page.

Press the ? key to see the modal.

Additions to original project:

  • Choose the target div via parameter
  • Choose which html to show in keyboard "popup"
  • Use the Jquery annotation:
    jQuery.questionmark(
    {
    'html_location' : '/question.mark.html',
    'target_id': 'helpTarget'
    }
    );
